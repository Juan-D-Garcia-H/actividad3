# Actividad 3 - Automatización de Base de Datos con Python

## 📌 Descripción

Este script automatiza la creación y el poblado de una tabla MySQL usando **SQLAlchemy** y **Faker**.  
Genera **100,000 registros** en una tabla llamada `personas_<tu_nombre>`.

## 🛠 Requisitos previos

- Python 3.10+
- MySQL instalado y corriendo localmente (XAMPP, WAMP, o servidor independiente)
- DBeaver para verificación visual
- Git (opcional, para clonar)

## 📦 Instalación

1. Clona este repositorio:
   ```bash
   git clone https://github.com/Juan-D-Garcia-H/actividad3.git
   cd actividad3
   ```
